# ThoughtFlow - منصة التفكير المرئي والتعلم العميق

![ThoughtFlow Logo](https://via.placeholder.com/600x200/667eea/white?text=ThoughtFlow)

## 🌟 نظرة عامة

**ThoughtFlow** هو منصة مجانية بالكامل للتفكير المرئي والتعلم التفاعلي، مصممة لتكون بديلاً محسّناً ومجانياً لـ Scrintal. يوفر ThoughtFlow لوحة لا نهائية للتفكير الحر، مع مساعد ذكاء اصطناعي مجاني ومميزات تعاونية متقدمة.

### ✨ المميزات الرئيسية

- 🎨 **لوحة لا نهائية**: فضاء غير محدود لترتيب أفكارك ومعلوماتك
- 🧱 **كتل متنوعة**: نصوص، صور، ملفات PDF، فيديوهات، وملاحظات سريعة
- 🔗 **روابط ذكية**: اربط الأفكار بأسهم وروابط ثنائية الاتجاه
- 🤖 **مساعد AI مجاني**: ذكاء اصطناعي متقدم للتلخيص والتحليل
- 👥 **تعاون فوري**: العمل مع الفريق في الوقت الفعلي
- 📋 **قوالب احترافية**: قوالب جاهزة للبحث والتعلم
- 📱 **تصميم متجاوب**: يعمل على جميع الأجهزة
- 🔒 **100% مجاني**: جميع المميزات متاحة مجاناً إلى الأبد

## 🚀 البدء السريع

### 1. فتح الموقع
```bash
# افتح index.html في المتصفح
open index.html
```

### 2. بدء التطبيق
- انقر على "ابدأ مجاناً الآن"
- أو انقر على "شاهد العرض التوضيحي" لمعاينة سريعة

### 3. استخدام اللوحة
- **التنقل**: اضغط Alt + اسحب للتحرك في اللوحة
- **التكبير**: استخدم عجلة الماوس للتكبير والتصغير
- **إضافة عناصر**: استخدم شريط الأدوات لإضافة نصوص وملاحظات
- **الربط**: استخدم أداة الربط لتوصيل العناصر

## 🛠️ البنية التقنية

### الملفات الرئيسية

```
webapp/
├── index.html          # الصفحة الرئيسية
├── styles.css          # ملف التصاميم
├── script.js          # الوظائف التفاعلية
└── README.md          # هذا الملف
```

### التقنيات المستخدمة

- **HTML5**: بنية الموقع الأساسية
- **CSS3**: تصاميم حديثة مع دعم الحركات والتأثيرات
- **JavaScript ES6+**: وظائف تفاعلية متقدمة
- **Fabric.js**: مكتبة الرسم على Canvas
- **Lucide Icons**: مجموعة أيقونات حديثة
- **Cairo Font**: خط عربي احترافي

## 🎯 الميزات المتقدمة مقارنة بـ Scrintal

### 🆓 مجاني بالكامل
- جميع المميزات متاحة مجاناً
- بلا حدود على عدد اللوحات أو العناصر
- مساعد AI مجاني (على عكس Scrintal المدفوع)

### 🤖 مساعد AI محسّن
- **تلخيص ذكي**: تلخيص النصوص والمقالات تلقائياً
- **ربط الأفكار**: اقتراح روابط بين المفاهيم المختلفة
- **تحليل المحتوى**: تحليل البيانات واستخراج الأنماط
- **اقتراحات تنظيمية**: تحسين ترتيب المحتوى بصرياً

### 🎨 تجربة مستخدم محسّنة
- **واجهة عربية**: دعم كامل للغة العربية
- **تصميم حديث**: واجهة أكثر جمالاً وسهولة
- **استجابة سريعة**: أداء محسّن على جميع الأجهزة
- **تحكم بديهي**: أدوات سهلة الاستخدام
